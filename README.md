# OS Experience Material Website

A simple website to provide study materials for Operating Systems.  
Get links to PDFs, books, and other resources hosted on Google Drive directly from the website.

## Features
- Easy access to OS study materials
- Clickable images linking to PDF files and books
- Simple and lightweight design for quick navigation

## How It Works
- Browse the website
- Click on images or links
- Open the corresponding PDF or resource from Google Drive

## Technologies Used
- HTML
- CSS
- JavaScript

## Usage
1. Clone or download the project
2. Open `index.html` in your browser
3. Click on images or links to access the study materials
